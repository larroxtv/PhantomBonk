## ⚡ Features

- 🚫 **Selective Phantom Control** — block phantom spawns by exact cause (`NATURAL`, `SPAWNER`, `SPAWNER_EGG`, `COMMAND`, `CUSTOM`) instead of one all-or-nothing toggle
- 🌍 **Per-World Rules** — override individual spawn causes for specific worlds, or exempt entire worlds completely
- 🔁 **Live Reload** — `/phantombonk reload` applies config changes instantly, no restart needed
- 🪶 **Lightweight** — one event listener, zero measurable performance impact
- 🔌 **Sensible Defaults** — blocks natural phantom spawns out of the box, fully configurable from there
- 🛡️ **Permission-Gated** — reload command locked behind `phantombonk.reload`

---

## 🛠️ Installation

1. Download the `phantombonk-VERSION.jar` file
2. Place it into your server's `/plugins` folder
3. Restart your server
4. Done — phantoms now follow your rules ✅
